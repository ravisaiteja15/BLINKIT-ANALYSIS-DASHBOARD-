# BLINKIT-ANALYSIS-DASHBOARD-

### 📌 Project Overview
This project presents an interactive Power BI dashboard built using Blinkit's sales and operational data. The dashboard provides insights into sales performance, customer preferences, outlet characteristics, and product distribution to support data-driven decision-making.

---
### 🎯 Objectives
- Analyze overall sales performance.
- Identify top-performing product categories.
- Understand customer rating trends.
- Compare outlet performance across different locations and sizes.
- Evaluate the impact of outlet type and establishment year on sales.

---
### 📊 Key Performance Indicators (KPIs)
- Metric	Value
- Total Sales	$1.20M
- Average Sales	$141
- Number of Items	8,523
- Average Rating	3.9
---

### 🛠️ Tools & Techniques Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Data Visualization
- Dashboard Design
---

### 📈 Business Impact
The dashboard helps stakeholders:
- Track sales performance efficiently.
- Identify profitable product categories.
- Compare outlet performance across regions.
- Make informed inventory and marketing decisions.

## 💡Conclusion

The Blinkit Analysis Dashboard provides a comprehensive overview of sales performance, product demand, customer preferences, and outlet efficiency. Through interactive visualizations and data-driven insights, the dashboard highlights key business trends such as top-performing product categories, outlet performance across different locations, and customer rating patterns.

The analysis reveals that outlet size, location tier, and product categories significantly influence sales performance. These insights can help businesses optimize inventory management, improve operational efficiency, and make informed strategic decisions to drive growth and enhance customer satisfaction.
